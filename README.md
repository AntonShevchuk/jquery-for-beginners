# jQuery for Beginners

A multilingual textbook about jQuery by [Anton Shevchuk](https://anton.shevchuk.name).

ISBN: 978-617-7150-93-9

## Available Languages

- [English](https://antonshevchuk.gitbook.io/jquery-for-beginners/en)
- [Українська](https://antonshevchuk.gitbook.io/jquery-for-beginners/uk)
- [Русский](https://antonshevchuk.gitbook.io/jquery-for-beginners/ru)

## Code Examples

Companion code examples are available as:
- Live demos: [anton.shevchuk.name/book/code](https://anton.shevchuk.name/book/code/)
- Source code: [github.com/AntonShevchuk/jquery-for-beginners-code](https://github.com/AntonShevchuk/jquery-for-beginners-code)

## Structure

Each language lives in its own directory (`ru/`, `uk/`, `en/`) and is published as a separate GitBook space. Shared assets (images, diagrams) are in `.gitbook/assets/`.
