# Про книгу

Цей підручник підійде як для початківців web-розробників, так і для досвідчених JavaScript-програмістів, які бажають освоїти нову для себе бібліотеку.

У підручнику прийнято наступне форматування:

**важливо** — запам'ятай, запиши, і завжди застосовуй

`<html>` — вихідний код у тексті виділяється monotype шрифтом

```javascript
// для лістингу вихідного коду буде використовуватись підсвітка синтаксису
function () {
  setTimeout("alert('Hello World!')", 5000);
}
```

важлива інформація, на яку слід звернути увагу
{% endhint %}

{% hint style="warning" %}
це точно важливо, не пропустіть
{% endhint %}

{% hint style="danger" %}
ну ви зрозуміли, так?
{% endhint %}

> замітка, яку варто взяти до відома, або, можливо, лише прочитати й посміхнутись, буде оформлена в блок тексту з відступом
