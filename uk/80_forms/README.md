# 80% Робота з формами

Форми – це, мабуть, один з найнелюбніших елементів на сторінці – поки налаштуєш зовнішній вигляд, потім ще перевір, що ввели недбалі користувачі, та виведи їм інформацію про допущені помилки, і, врешті-решт, відправляєш на сервер дані з почуттям полегшення від виконаної кропіткої роботи. Так ось – про те, що допоможе в цій самій роботі, я й буду розповідати далі.
