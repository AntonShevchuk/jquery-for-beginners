# 0% Про HTML, CSS та JavaScript

Почнемо знайомство з jQuery з повторення (або вивчення) основ правильного вживання зв'язки HTML та CSS з невеличкою домішкою JavaScript.

> Якщо не хочете впасти в бруд обличчям перед колегами — не пропускайте цю главу «повз вуха».

HTML — про нього варто пам'ятати дві речі — **семантичний** та **правильний**.
