# Розділяй та владарюй

Тут варто запам'ятати декілька простих правил:

* виносимо JavaScript у зовнішні файли
* ніколи не пишемо inline обробників подій `onclick="some()"`
* виносимо CSS у зовнішні файли
* ніколи не пишемо inline стилів `style="color:red"`
* і ще разок для закріплення — ніколи не пишемо inline!

Тепер наведу код, за який слід щось ламати (це приклад поганого коду, уточнюю для тих, хто не розуміє натяків):

```markup
<script>
function doSomething(){ /* … */ }
/* чується хрускіт зламаних кісток зап'ястя, щоб не друкував */
</script>
<style>
p { line-height:20px; }
/* крхххх… гомілкова кістка, і на роботу вже не піде */
</style>
<div style="color:red;font-size:1.2em">
    <p onclick="doSomething();">Lorem ipsum dolor sit amet...</p>
    <!-- тидищ, головою об стіл… насмерть, як жест милосердя -->
</div>
```

Незрозуміло, чому ж це погано? Схоже, тобі просто не доводилося міняти дизайн для вже готового сайту :) Проясню суть проблеми: тобі ставлять задачу — «_треба поміняти колір шрифту для всіх сторінок сайту_», яких може бути три десятки. Це можуть бути не лише HTML-файли, а сторінки якогось [шаблонізатора](https://ru.wikipedia.org/wiki/%D0%A8%D0%B0%D0%B1%D0%BB%D0%BE%D0%BD%D0%B8%D0%B7%D0%B0%D1%82%D0%BE%D1%80), розкидані по двох десятках тек (і це ще не найгірший варіант). І тут з'являється він — червоний абзац. Імовірність почути «слова підтримки» на адресу автора цього коду буде прагнути до одиниці. Щодо inline-обробників подій ситуація схожа. Ось уяви собі — пишеш ти JavaScript-код, усе чудово, усе виходить, поки ти не намагаєшся клацнути по червоному абзацу. Він виявляється тобі не підвладний і живе своїм власним життям, ігноруючи всі твої зусилля. Ти дивишся код, і знову хтось почує ці слова...

При застосуванні чотирьох правил «червоного абзацу» у тебе має вийти чистий і передбачуваний HTML код:

```markup
<div id="abzac">
    <p>Lorem ipsum dolor sit amet...</p>
</div>
```

Стилі можна буде легко повісити на `<div>` з ідентифікатором, як, власне, і обробник подій для нашого параграфа.

> Абзац — не параграф, але для красного слівця та легкості засвоєння згодиться
