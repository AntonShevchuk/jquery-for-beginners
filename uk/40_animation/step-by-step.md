# Крок за кроком

Хотілося б окремо зупинитись на функції `step`, і для наочності наведу приклад реалізації подібної функції, яка відображає поточне значення анімованого параметра:

```javascript
// елемент, який буде використовуватися для виведення інформації
let $h2 = $("h2");
// наша step-функція
let customStep = function(now, fx) {

    fx.elem;    // DOM-елемент, який анімується
    fx.prop;    // параметр, який анімується
    fx.start;   // початкове значення
    fx.end;     // кінцеве значення
    fx.pos;     // коефіцієнт, змінюється від 0 до 1
    fx.options; // опції налаштувань анімації
    now;        // поточне значення анімованого параметра, обчислюється як
                // now = (fx.end - fx.start) * fx.pos

    $h2.html(`${fx.prop}: ${now}${fx.unit}`); // виведення тексту
};
// викликаємо анімацію з довільною функцією
$("img").animate({ height: "-=10px" }, { step: customStep });
```

{% hint style="warning" %}
Варто запам'ятати, що step-функція буде запускатися для кожного елемента у вибірці та для кожної властивості, яку ми анімуємо.
{% endhint %}

За допомогою даної функції можна зупинити анімацію при досягненні заданих значень:

```javascript
// step-функція, яка зупиняє анімацію
// при значенні 'left' меншому або рівному нулю
let limitForLeft = function (now, fx) {
  if (fx.prop === "left" && fx.start > fx.end && now <= 0) {
    $(fx.elem).stop() 
  }
}
```

> Мені жодного разу не доводилося використовувати step-функції в комерційних проєктах, лише для прикладів коду в цьому підручнику.
