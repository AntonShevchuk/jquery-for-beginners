# Будь готовий!

Тепер час братися до роботи — візьмемо якийсь елемент на сторінці й спробуємо його змінити. Для цього в `<head>` вставимо наступний код:

```markup
<script>
    // ми намагаємося знайти всі елементи <h2> на сторінці
    // та змінити колір шрифту на червоний
    jQuery("h2").css("color", "red");
</script>
```

Тільки подібний код нічого не зробить, оскільки на момент виконання на сторінці ще не буде тегів `<h2>` – надто рано виконується скрипт, до завантаження всього HTML-документа. Для того щоб код спрацював правильно, ми повинні або помістити код після шуканого `<h2>`, а то й одразу в самий низ сторінки, або використати метод `ready()` для відстеження події `DOMContentLoaded` нашого `document`:

```markup
<script>
    // чекаємо завантаження всього документа
    // після цього буде виконана анонімна функція
    // яку ми передали як параметр
    jQuery(document).ready(function(){
        jQuery("h2").css("color", "red");
    });
</script>
```

Також можна використати скорочений варіант без явного виклику методу `ready()`:

```markup
<script>
    $(function(){
        $("h2").css("color", "red");
    });
</script>
```

{% hint style="warning" %}
Останній варіант уже варто вважати основним для використання, адже в jQuery 3.0 метод `ready()` вже позначений як deprecated.
{% endhint %}

Ви можете створити скільки завгодно багато подібних функцій, не обов'язково весь необхідний функціонал поміщати в одну.

{% hint style="info" %}
Якщо ваш код додано на сторінку та він буде виконуватися вже після настання події `load`, то він буде викликаний негайно, жодних додаткових маніпуляцій виконувати не потрібно.
{% endhint %}

`$()` — це синонім для `jQuery()`. Щоб у вас не виникало конфліктів з іншими ~~країнами~~ бібліотеками за використання «`$`», раджу ваш код обгортати в анонімну функцію наступного вигляду (best practice, так би мовити):

```javascript
;(function($, undefined){
    // тут тихо й затишно
    // ми завжди будемо впевнені, що $ === jQuery
    // a undefined не перевизначений ;)
})(jQuery);
```
