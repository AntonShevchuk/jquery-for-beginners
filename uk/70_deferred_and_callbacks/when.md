# When

Тепер на прикладі нашої [підопічної сторінки](https://anton.shevchuk.name/book/code/when.html) покажу хитрий метод `$.when()`:

```javascript
$.when(
  $("article h3").hide(3000),
  $("article img").slideUp(1000),
  $("article p").fadeTo(2000, 0.1)
).then(() =>
  $("article").slideUp(500)
).done(() =>
  console.info("done")
).catch(() =>
  console.error("fail :(")
)
```

Поясню, що відбувається – всі анімації стартують одночасно, і коли всі завершать свою роботу, буде викликано функцію, яку ми передаємо як аргумент у метод `then()` (одну з двох, залежно від результату того, що відбувається).&#x20;

Для забезпечення роботи цієї «магії» методи `$.when()`, `$.ajax()` та `animate()` реалізують інтерфейс Deferred. Тому ми можемо використовувати всі ті методи, з якими знайомилися в попередньому розділі.

{% hint style="info" %}
Тепер можна і по-розумному – метод `when()` повертає проєкцію Deferred об'єкта, приймає як параметри довільну множину Deferred об'єктів, коли всі вони відпрацюють, об'єкт `when` змінить свій стан на «виконано», з подальшим викликом усіх підписантів.
{% endhint %}
