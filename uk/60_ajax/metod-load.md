# Метод load()

Почнемо з найпростішого — завантаження HTML-коду в потрібний нам DOM-елемент на сторінці. Для цієї мети нам підійде метод `load()`. Даний метод може приймати наступні параметри:

<table data-header-hidden><thead><tr><th width="213">аргумент</th><th>опис</th></tr></thead><tbody><tr><td><code>url</code></td><td>запитуваної сторінки</td></tr><tr><td><code>data</code></td><td>дані, що передаються (необов'язковий параметр)</td></tr><tr><td><code>callback</code></td><td>функція, яка буде викликана при завершенні запиту до сервера (необов'язковий параметр)</td></tr></tbody></table>

Тепер на прикладах. Відкрий підопитну сторінку:

За допомогою наступного запиту, в елемент з `id="content"` буде вставлено весь HTML із вказаної сторінки:

```javascript
// в елемент з id="content" буде вставлено HTML із вказаної сторінки
$("#content").load("example.html");
```

Як бачиш, підвантажується весь HTML разом з усіма стилями. Щоб уникнути подібного безладу, з підвантажуваної сторінки слід вибрати лише потрібний нам елемент. Для цього достатньо через пробіл вказати селектор:

```javascript
// в елемент з id="content" буде вставлено HTML із вказаної сторінки,
// але не весь, а лише вибраний за вказаним селектором тег main
$("#content").load("example.html main");
```

По закінченні завантаження ми можемо викликати довільну функцію для обробки підвантажуваного HTML. Ось тільки дана функція спрацює лише після того, як контент буде додано до цільового елемента, а мені б хотілося цим процесом керувати:

```javascript
// обробляємо отримані дані
$("#content").load("example.html main", function(data) {
    // нічого оригінальнішого не придумав
    alert("Ми завантажили HTML довжиною в " + data.length + " символів");
});
```

Якщо тобі треба передати на сервер якісь дані, то це можна легко зробити наступним чином:

```javascript
// передаємо дані на сервер другим параметром
// от тільки сервер наш їх ніяк не обробляє
// але ти можеш побачити дані в консолі, у відправлених запитах
$("#content").load("example.html main", { id: 42 });
```

> З мого досвіду роботи — тобі дуже часто доведеться користуватися методом `load()` як описано в першому прикладі. А ще раджу запам'ятати другий приклад, він може виручити, коли треба реалізувати завантаження через AJAX, а доступу до сервер-сайду у тебе немає або він обмежений.
