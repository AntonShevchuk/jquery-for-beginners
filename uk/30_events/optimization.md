# Оптимізація

Неявним бонусом від використання «живих» подій можна вважати можливість оптимізації. Про «підхожі» випадки я й розповім.

Випадок перший, банальний – уявіть собі таблицю на тисячу рядків та десяток стовпців, а тепер спробуйте підрахувати, скільки пам'яті з'їдять обробники події `click` для кожної комірки?

```javascript
$("table td").on("click", function() { 
  /* ... */
});
```

Ось і я гублюся в здогадках. Але зате точно знаю, як варто його переписати для економії пам'яті та часу:

```javascript
$("table").on("click", "td", function() { 
  /* ... */
});
```

Випадок другий, надуманий – необхідно записувати дії користувача на сторінці, тобто відстежувати кліки по незліченній кількості об'єктів:

```javascript
$("body").on("click", "*", function() {
    console.info("Click on "+this.tagName);
});
```
