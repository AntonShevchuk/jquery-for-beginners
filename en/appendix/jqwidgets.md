# jQWidgets

[jQWidgets](https://www.jqwidgets.com/) is not just a library, it's a whole combine harvester packed with useful widgets. There are currently more than 60 of them, so just listing them all with a screenshot for each would fill half a book. So we'll make do with a brief summary of a few widgets — hopefully that'll be enough to whet your appetite and get you to check out the developer's website.

*   jqxGrid — this is a data grid packed with useful (and not so useful) features. It's got pagination, sorting, filtering, and grouping. You could write a massive tome just about this widget alone:



    <figure><img src="../../.gitbook/assets/jqxGrid.png" alt=""><figcaption></figcaption></figure>
*   jqxChart — a widget for building all kinds of charts using HTML, CSS, and JavaScript. Everything is done very, very nicely — I especially want to highlight the save-chart-as-image feature, which is sometimes sorely missing:



    <figure><img src="../../.gitbook/assets/jqxChart.png" alt=""><figcaption></figcaption></figure>
*   jqxGauge — you won't find this widget often even in more well-known frameworks, but essentially it's a measurement instrument — you can use it to draw a speedometer, pressure gauge, thermometer, or any other measuring device with a custom scale:



    <figure><img src="../../.gitbook/assets/jqxGauge.png" alt=""><figcaption></figcaption></figure>
*   jqxTreeMap — another rare species, unique even. It lets you build a linked tree as organized rectangles. If that makes no sense, better check out the [demo](https://www.jqwidgets.com/jquery-widgets-demo/demos/jqxtreemap/index.htm#demos/jqxtreemap/defaultfunctionality.htm), and here's a screenshot:



    <figure><img src="../../.gitbook/assets/jqxTreeMap.png" alt=""><figcaption></figcaption></figure>
*   jqxTree — this one is not as exotic; as the name suggests, we'll be planting trees: &#x20;

    <img src="../../.gitbook/assets/jqxTree.png" alt="jqxTree" data-size="original">

That wraps up the "cool" widgets overview. I don't feel like diving into the boring, mundane form element wrappers. I'll just note that in many ways this framework surpasses jQuery UI, but not everything is rosy in this kingdom:

{% hint style="warning" %}
This framework is only partially free — it does have a [Community license](https://www.jqwidgets.com/license/), but in practice, you'll still need to open your wallet if you need the widgets.
{% endhint %}

It's also worth mentioning one nice feature — easy integration with various JavaScript frameworks. But that's another story.
