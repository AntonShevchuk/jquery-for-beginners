–¢–µ–ø–µ—Ä—å –ø—Ä–∏—Å—Ç—É–ø–∏–º –∫ CSS, –∏ –Ω–∞—á–Ω—ë–º, –ø–æ–∂–∞–ª—É–π, —Å —Ä–∞—Å—à–∏—Ñ—Ä–æ–≤–∫–∏ –∞–±–±—Ä–µ–≤–∏–∞—Ç—É—Ä—ã CSS. –≠—Ç–æ Cascading Style Sheets, –¥–æ—Å–ª–æ–≤–Ω–æ ¬´–∫–∞—Å–∫–∞–¥–Ω–∞—è —Ç–∞–±–ª–∏—Ü–∞ —Å—Ç–∏–ª–µ–π¬ª, –Ω–æ:
